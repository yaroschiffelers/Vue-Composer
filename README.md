# Vue Composer

> Compose and structure a view component based on some data structure.

**This project is still a work in progress**

## Installation

As package:

```bash
yarn add vue-composer
```

```javascript
// in main.js
import VueComposer from 'vue-composer'

Vue.use(VueComposer)
```
